# Serverless REST API

This is a didactic demonstration about how to create a serverless TodoApp.

Visit https://serverless.com/ for more info.

## Setup

- Create an AWS account
- Set up AWS CLI
- Run `npm -g i serverless`
- Run `npm i`

## Deploy

```bash
serverless deploy
```

## Remove

```bash
serverless remove
```

## Redeploy a function

```bash
serverless deploy function -f functionName
```
